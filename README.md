# FUTURE_FS_03
# 🛒 Departmental Store Website

A professional and responsive website built for a local departmental store to enhance its online presence and attract more customers.

---

## 📌 Project Overview

This project is designed to help a local business:

* Build a strong online presence
* Provide easy access to products and services
* Improve customer trust and engagement
* Enable quick contact via call and WhatsApp

---

## 🎯 Objective

To create a clean, modern, and user-friendly website that showcases:

* Store information
* Available products
* Contact details
* Easy communication options

---

## 🚀 Features

* 🌙 Dark theme modern UI
* 🎴 Flash card (flip card) product display
* 📱 Mobile-friendly responsive design
* 📞 One-click call button
* 💬 WhatsApp integration
* ⚡ Fast and lightweight

---

## 🧰 Technologies Used

* HTML
* CSS
* JavaScript

---

## 🏗️ Website Sections

* **Home** – Introduction to the store
* **Products** – Flash card style product categories
* **About** – Business information
* **Contact** – Phone, location, and WhatsApp

---

## 💡 Problem Solved

Many local departmental stores do not have a website, making it difficult for customers to:

* Find store details online
* Know available products
* Contact the store easily

This website solves these problems by providing a simple and accessible digital platform.

---

## 📈 How It Helps the Business

* Increases visibility on the internet
* Builds trust with customers
